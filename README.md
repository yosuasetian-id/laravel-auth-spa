<h1 align="center">Laravel SPA Authentication</h1>

<p align="center">A clean and modern Laravel project that implements Single Page Application (SPA) Authentication using Laravel Sanctum.</p>

<hr/>

## Tech Stack
```text
1. Laravel 12
2. Laravel Sanctum
3. MySQL / PostgreSQL (PostgreSQL)
4. REST API
5. SPA FrontEnd (Panding)
6. Mail Service
```

## Features
```text
✅ Register new users
✅ Email verification after registration
✅ Secure login system
✅ Logout from authenticated session
✅ Forgot password via email
✅ Reset password with secure token
✅ SPA Authentication using Sanctum (Cookie-based)
```
